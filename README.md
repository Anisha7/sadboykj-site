## How to run the app:
    1. Clone the repository.
    2. Open your terminal
    3. `cd api`
    4. `npm install`
    5. `nodemon server.js`
    6. Open a new terminal or new tab in your terminal
    7. `cd client`
    8. `npm install`
    9. `npm start`

    This should run the website!
